This is a dynamic blog viewer web application that fetches and displays blog content using external APIs. The application demonstrates the use of JavaScript for DOM manipulation and asynchronous data fetching to create an interactive user interface.

🔹 Features

-Fetches blog data from external APIs
-Dynamically updates content using JavaScript
-Responsive UI built with Bootstrap
-Displays multiple blog posts with images and descriptions
-Clean and structured layout

🔹 Tech Stack

-HTML5
-CSS3
-Bootstrap
-JavaScript

🔹 Concepts Used

-API Integration (Fetch API)
-DOM Manipulation
-Asynchronous Programming
-Responsive Design

🔹 How to Run

1.Clone the repository
2.Open the project folder
3.Run the index.html file in your browser
